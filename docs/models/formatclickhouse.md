# FormatClickHouse

Data format to use when sending data to ClickHouse. Defaults to JSON Compact.


## Values

| Name                               | Value                              |
| ---------------------------------- | ---------------------------------- |
| `JSON_COMPACT_EACH_ROW_WITH_NAMES` | json-compact-each-row-with-names   |
| `JSON_EACH_ROW`                    | json-each-row                      |
